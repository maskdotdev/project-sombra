# Project Sombra

Project Sombra is an embedding-focused project for fast, high-throughput code search.

## What is in this repo
- Training pipeline for a from-scratch code embedding model
- Hard-negative mining workflow
- ONNX export + quantization utilities
- Retrieval benchmarking scripts
- Iteration notes and progress docs in `docs/`

## Quick start
```bash
uv sync
uv run python scripts/check_wsl_gpu.py
```
